Portfolio Fritz – Flask + MySQL

Este es un portfolio web dinámico desarrollado con Flask, MySQL, HTML/CSS/JS, Jinja2 y POO.
Incluye un panel administrador para gestionar información personal, proyectos, experiencia, educación y habilidades.

Tecnologías utilizadas

Python – Flask

MySQL (Clever Cloud)

HTML5, CSS3, JavaScript

Jinja2

Flask-WTF

Bootstrap

Git + GitHub

Características Principales

Sistema de login seguro

Hash de contraseñas con werkzeug.security

Validación de formularios con Flask-WTF

Panel administrador dinámico

CRUD completo para todas las secciones

Animaciones y transiciones visuales

Diseño responsive (celular / tablet / escritorio)

Iconos de FontAwesome

Google Fonts

Iframe integrado

Base de Datos

El proyecto utiliza una base de datos MySQL con las siguientes tablas:

usuarios

datos_personales

experiencia

educacion

habilidades

proyectos

El archivo portafolio_db.sql contiene el modelo físico completo.

Instalación local
pip install -r requirements.txt
python app.py


Luego abrí:

http://localhost:5000
